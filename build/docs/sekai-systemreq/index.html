System Requirements

To run a sekaid application the following system requirements must be met:

Hardware​

  • 4 vCPU cores (ARM64 or x64)
  • Minimum 8GB of RAM
  • Storage space required to persist blockchain state and snapshots (1TB+ recommended)
  • Minimum 32GB+ of free storage space available at all times

Networking​

  • Stable internet connection with minimum 64 Mbps Up/Dn speed at all times
  • Static IP address or dynamic DNS
  • Access to router or otherwise your local network configuration

Software​

  • Ubuntu 20.04 LTS installed on the host instance or VM
  • Secure SSH configuration with RSA key or strong password
