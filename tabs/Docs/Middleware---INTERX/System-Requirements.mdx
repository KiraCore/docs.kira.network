To run the `interxd` application the following system requirements must be met:


## Hardware {#409839eed65943ba928a73c6843e3082}

- `1 vCPU` core (ARM64 or x64)
- Minimum `4 GB` of RAM
- Storage space required to cache  (`256 GB+` recommended)
- Minimum `32 GB+` of the **free** storage space available at all times

## Networking {#37b99c5f80ab438dbeb94966ed21b251}

- Stable internet connection with minimum `64 Mbps` Up/Dn speed at all times
- Static IP address or dynamic DNS (DDNS)
- Access to router or otherwise your local network configuration
- Exposed port `11000`

## Software {#d7ff0322b24a4d31851b072779d27c2f}

- Ubuntu `20.04 LTS` installed on the **host** instance or VM
- Secure SSH configuration with RSA key or strong password
- Pre-installed latest release of docker
